## Overview
Dragon News is a modern, responsive news web application where users can browse news articles by category. The project uses **React** for the frontend, **React Router** for dynamic page navigation, and **TailwindCSS** for sleek, responsive design.  



---

## Features
- Dynamic routing using **React Router**  
- Browse news by category (e.g., sports, technology, business)  
- Responsive design for mobile and desktop  
- Clean and modern UI for reading news  
- (Optional) User authentication via Firebase  

---

## Tech Stack
- **Frontend:** React, React Router, TailwindCSS  
- **Backend/Database:** Firebase (optional)  
- **HTTP Requests:** Axios  

---

## Dependencies
- react  
- react-router-dom
